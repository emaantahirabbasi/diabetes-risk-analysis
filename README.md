# 🩺 Diabetes Risk Factor Analysis

## Overview
Exploratory data analysis on the Pima Indians Diabetes Dataset 
(768 patient records) to identify the strongest predictors of 
diabetes onset. Led as a semester project with a team of 4.

## 📊 Key Findings
- Glucose, BMI, and Age identified as top 3 risk factors
- Patients with glucose > 140 showed significantly higher diabetes rates
- 5+ professional visualizations produced for clinical interpretation
- Actionable recommendations delivered for early screening

## 🛠️ Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn

## 📁 Dataset
Pima Indians Diabetes Database — [Kaggle Link](https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database)
